Java在最初的多线程实现程序Thread、Runnable之中，是没有返回值的。这样的接口带来了很多处理多线程程序的不便。所以后来又引入了Callable接口，以及与之对应的Future接口。这样我们的多线程程序提交到线程池执行的时候就可以返回一个Future对象。然后调用Future.get()就可以获取线程的返回值。
